📚 Study Planner — Python Tkinter App

A modern, feature-rich desktop study planner built using Python and Tkinter.
Designed to help students stay organized, track progress, and boost productivity with built-in tools like a Pomodoro timer.

⸻

✨ Features
	•	✅ Add, edit, and delete study tasks
	•	🎨 Subject-based color tagging
	•	⚡ Priority levels (High / Medium / Low)
	•	✔️ Mark tasks as completed
	•	⏳ Built-in Pomodoro Timer
	•	📊 Progress dashboard
	•	🔍 Search and filter tasks
	•	💾 Persistent data storage using JSON
	•	🖥️ Clean and modern UI (dark theme)

⸻

🛠️ Tech Stack
	•	Python 3
	•	Tkinter (GUI framework)
	•	JSON (data storage)
	•	Datetime module
  Tasks are stored in a local JSON file
	•	Each task includes:
	•	Subject
	•	Topic
	•	Due date
	•	Priority
	•	Notes
	•	Completion status
	•	The app dynamically updates UI components like:
	•	Task list
	•	Dashboard
	•	Timer

⸻

⏱️ Pomodoro Timer
	•	Work Session: 25 minutes
	•	Short Break: 5 minutes
	•	Long Break: 15 minutes

Helps maintain focus and prevent burnout.

⸻

📊 Dashboard

Track your productivity with:
	•	Total tasks
	•	Completed tasks
	•	Pending tasks
	•	Pomodoro sessions completed
